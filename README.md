### 🧠 Prompt Engineering Techniques Used

- **Persona prompting** — instructed Claude to write as a UGC creator, not a brand
- **Format constraints** — specified Hook → Story → Solution → CTA structure
- **Tone anchoring** — separate tone instructions per platform (aesthetic, conversational, energetic)
- **Audience targeting** — defined age group, location, and platform in the prompt
- **Output formatting** — requested director notes, strategy rationale, and timing per script
